# Open Command In Default Editor

While using the shell you can invoke your default editor (`$EDITOR`) to write or edit a command.

Just type <kbd>Ctrl</kbd>+<kbd>X</kbd> <kbd>Ctrl</kbd>+<kbd>E</kbd>.
