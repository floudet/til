# Sort Lines

Vim has a very powerful built-in sort utility. It can be invoked as follows:

```
:{range}sort
```


**Examples:**

Sort alphabetically, removing duplicate lines on the whole file

```
:%sort u
```

Numeric sort

```
:sort n
```

Reverse sort

```
:%sort!
```
